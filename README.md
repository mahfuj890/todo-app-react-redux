 - add todo

 - filter todo

## Installation

first run the api server, go to the "json-server-todos" folder and open the cmd or terminal 

```bash
  npm install 
  npm start
```
then run the  react, go to the "todo-app-react-redux" folder

```bash
  npm install 
  npm run dev
```
## Tech Stack

**Client:** React, Redux, TailwindCSS

**Server:** Node, Express
